This folder contains the first population of the database with areas. It is used to populate the database with the initial data for the areas. The data is stored in a JSON file and is imported into the database.
The process have to be run only once at the beginning of the project, to populate the database with the initial data for the areas. After that, the data can be updated manually or through other means as needed.

THE STRUCTURE OF THE JSON ENTITY IS COHERENT WITH THE AREA ENTITY DEFINED IN THE AREA SERVICE.
